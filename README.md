# Github Finder

A simple React application that helps you find users on Github.

## Screenshot

![home view](https://github.com/King-Benx/Github-Finder/blob/master/screenshot/screenshot.png)

## Commands

### Build Application and its dependencies

Run `yarn`

### Run application

Run `yarn start`
